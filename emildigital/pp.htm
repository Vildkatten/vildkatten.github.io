Project Women's World

The brief was to find new ways to get women to join the army, and this is the result me and my collegue Maria Rundqvist came up with. Our approach was to create an object, the orb, that would link the user with other women from the armies around the world as well as for the user to get enlightened. Itâ€™s basically a information system using an orb as a remote control. The system uses arduino microcontrollers, wireless Xbee shield, accelerometer and Processing software.

The briefing box

Mission briefing system. You log on to the system using a RFID card. Doing this you will see your assignment on the screen. Application will be terminated in 15 seconds.

Milliwear

Application that suggests appropriate clothing and accessories, depending on temperature and wind speed, using live feed from pachube.com.

The Ping Jacket

The Ping Jacket is a jacket that senses when somebody is coming at you from behind. An ultrasound sensor is stitched in the back of the jacket, and the lights on the arm indicates the distance between you and your encounter. Use for spying purposes and whenever you need "a eye in the back".

Tracko - A tracking device

Cool device tracking a colored cloth using Processing client, webcam and arduino board.

Motion-Tec

Motion detection using ultrasound sensor. Communication between sensor and actuator (arduino to arduino).
